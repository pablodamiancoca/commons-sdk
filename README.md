# Heartland & Global Payments - Commons SDK

Common functionality to be reused by Global Payments SDKs.

## Installation

Installing it into your solution is usually be done by using Maven.

To install via [Maven](https://maven.apache.org/guides/introduction/introduction-to-dependency-mechanism.html):

```
<dependency>
  <groupId>com.heartlandpaymentsystems</groupId>
  <artifactId>globalpayments-sdk</artifactId>
  <version>1.4-SNAPSHOT</version>
</dependency>
```
